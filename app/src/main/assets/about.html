About the App

Android Tips App provides you all the tips and tricks required for a new Android User. You can use these tips to customize your phone as per your needs.
This application is an offline application. You can see the content even when you are not connected to the internet. This application downloads data for offline usage.

About Hanu-Droid Framework

This application is developed using the Hanu-Droid framework. The Hanu-Droid framework allows you to develop android applications within few minutes. You need not worry about coding, just concentrate on content creation.
